# Zomato-Revenue-Operation-Analytics-
# 🍽️ Zomato Revenue & Operations Analytics

An end-to-end Data Analytics project focused on analyzing Zomato's food delivery ecosystem using **Excel, SQL, Python, and Power BI**. The project uncovers valuable insights into revenue trends, customer behavior, restaurant performance, cuisine preferences, and delivery operations through data cleaning, advanced analytics, and interactive dashboarding.

---

## 📌 Project Overview

The food delivery industry generates large volumes of transactional and operational data. This project leverages customer, restaurant, menu, order, and delivery datasets to identify business opportunities, operational bottlenecks, and customer behavior patterns.

The objective was to transform raw data into actionable business insights through a complete analytics workflow involving data cleaning, SQL analysis, Python-based analytics, and Power BI dashboard development.
